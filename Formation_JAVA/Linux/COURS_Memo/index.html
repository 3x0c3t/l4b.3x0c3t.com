Commandes Linux par Thématique

Dossiers

  • ls Liste les fichiers et répertoires
  • cd Change de répertoire
  • mkdir Crée un nouveau répertoire
  • rmdir Supprime un répertoire vide
  • pwd Affiche le répertoire courant

Fichiers

  • cat Affiche le contenu d'un fichier
  • cat file1 file2 Affiche le contenu de plusieurs fichiers
  • cat *.txt Affiche le contenu de tous les fichiers texte
  • touch Crée un fichier vide ou met à jour la date d'un fichier
  • nano Éditeur de texte en ligne de commande
  • vi Éditeur de texte avancé
  • less Affiche le contenu d'un fichier page par page

Recherche

  • find Recherche des fichiers et répertoires
  • grep Recherche des chaînes de caractères dans un fichier
  • locate Trouve rapidement des fichiers par nom
  • which Trouve l'emplacement d'un exécutable
  • whereis Trouve les fichiers binaires, sources et de manuel

Modifications

  • cp Copie des fichiers et répertoires
  • mv Déplace ou renomme des fichiers et répertoires
  • rm Supprime des fichiers ou répertoires
  • chmod Change les permissions des fichiers
  • chown Change le propriétaire des fichiers

Scripts

  • bash Lance un script bash
  • sh Lance un script shell
  • ./script.sh Exécute un script local
  • crontab Programme des tâches automatiques
  • at Programme une commande à exécuter plus tard

SSH

  • ssh Se connecte à un serveur distant via SSH
  • scp Copie des fichiers entre hôtes via SSH
  • ssh-keygen Génère des clés SSH
  • ssh-agent Démarre un agent pour les clés SSH
  • ssh-add Ajoute une clé privée à l'agent SSH
